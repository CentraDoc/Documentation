◆ MultiRIPPageCB

typedef int(* MultiRIPPageCB) (void *p, CDocRIP *rip, int pageN0, char *oFileName, RIPTargetSpec *uSpec)

MultiRIP autorun page callback - at start of page.

pass the page and return a filename and targetspec filename is only relevant if memory mapped file, and useful if raw return true to re-spec the target, false to continue with the same target

not guarded and called on arbitrary threads

◆ MultiRIPStoreCB

typedef int(* MultiRIPStoreCB) (void *p, int pageN0, CDocTarget *target)

MultiRIP autorun store callback - at end of page target is complete, store it as appropriate return 0 return of !0 is reserved and currently ignored not guarded and called on arbitrary threads.
