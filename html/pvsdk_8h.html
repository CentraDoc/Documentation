Detailed Description

PreviewSDK - simplified DLL interface

The PreviewSDK is a simplified API for rendering PostScript and PDF files using the CentraDoc Windows GDI driver.

PreviewSDK also has support for rendering EMF files.

Set special parameters.

This feature has been merged with the ini file mechanism. See the ini file documentation for more parameters: pvsdkini.txt

These settings are no longer specific to the handle, but global to the whole library. handle is now ignored and may be zero (signature change is pending).

Parameters should be set before any renderings is done.

Parameter names and values are separated by = and parameter sets by
For example: 

    "level2patterns=Y\nAlwaysUseEmbeddedFonts=Y"

handleIGNORED

These functions grant access to the Pdf API via the GCO Class mechanism. Refer to pdf.d for more information.

this kind of implies a need for sysframe TRY/CATCH support, since accessing the PDF api bypasses any intermediate wrapping. pvsdkl.c can do this transparently; the import lib will need some attention.

int LS_API LS_AddFontMap ( LsHandle  handle,
const char *  psFont,
const char *  winFont 
)

Add a font mapping.

Returns
non zero for success or zero for failure

NOTES:

  • The global font map survives the creation and deletion of individual files, but does not survive the exiting and restarting of the program.
  • The local font map gets created when you open a file and at that point is a duplicate of the global font map.
  • Changes to the global font map will not affect changes to any local font maps inside existing open files, but only when a file is opened.
  • The initial global font map on startup is as follows:

Times-Italic Times New Roman Italic
Times-Bold Times New Roman Bold
Times-BoldItalic Times New Roman Bold Italic
Helvetica Arial

Helvetica-Bold Arial Bold
Helvetica-BoldOblique Arial Bold Italic
Courier Courier New
Courier-Oblique Courier New Italic
Courier-Bold Courier New Bold
Courier-BoldOblique Courier New Bold Italic
Symbol Symbol
ZapfDingbats Monotype Sorts
Parameters
handleA handle to an open file for the local font map; if NULL, it looks at the global font map
psFontThe PostScript font name to map
