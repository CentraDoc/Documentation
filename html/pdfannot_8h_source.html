31 This means not all methods apply to all instances.
32
33 A link may represent:
34 an internal link to a specific destination (page, location, view state)
35 an external link to a pdf w/specific destination
36 resolving the destination _may_ require opening the external pdf
37 and passing it to the link for name resolution
38 a url
39 IsMap and Base are url specific
40 a file attachment
41 which can be opened and read from
42 an external file or program reference
43 non-pdf, non-url, non-attached, etc.
44
45 GetTarget returns the external file name or url
46 an internal link returns 0
47 ?length is returned in case of DBCS or Unicode encodings.
48
49 HasDest returns 0 for non-pdf link, 1 for resolved link, -1 for needs resolving
50 GetPage returns the page destination (0 based, -1 for none/unresolved)
51 GetDest returns details about location, view state, zoom
52 ResolveLink is used to resolve an external link
53
54 HasAttachment returns 1 if there is an embedded file.
55 OpenAttachment returns a FFsPtr to read the embedded file.
56
57 IsMap returns 1 if the URL is to be treated as an image map, passing x,y:
