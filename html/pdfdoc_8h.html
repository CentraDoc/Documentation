◆ PdfAccessFlags

enum PdfAccessFlags

Symbols for permissions returned by: Authorize, CurrentPermissions and GetUserPermissions All undefined flags must be set to 1, except bit 1&2 are set to 0.

this makes all valid values negative integers *

Note that not all combinations are valid, only some are supported by Acrobat. Enum is reorganized to highlight this: select a print option select a changes option optional content copying flag always add in the accessibility flag (according to PDF 2.0)
