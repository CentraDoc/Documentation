50 freetype uses limits and does some mucking about to deal with int == 2
51
52 long long should be dealt with here

55/* unfortunately compilers don't process `sizeof` in preprocessor
56 _WIN64 is a given, but gcc is trickier. lots of possible options;
57 it seems that long is defined as 8 bytes under gcc 64, so we'll try that.
58
59 apparently __LP64__ means both long and void * are 8 bytes,
60 but _WIN64 a long is still 4 bytes.
61
62 thus, if you need to do pointer <-> integer stuff you'll need
63 ptrdiff_t, and ptrdiff_t is signed, so beware.
64
65 maybe we need ptrequiv_t, although the math is still weird.
66*/
67
